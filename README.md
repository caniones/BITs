# BITs - Bitácora de Ingresos y TrabajoS 🛠️💻

**BITs** es un sistema de escritorio diseñado exclusivamente para optimizar, organizar y acelerar la atención en talleres de servicio técnico de computadoras (PC y Notebooks). 

Funciona como un asistente digital que registra el historial de cada equipo desde que ingresa al taller hasta que se entrega reparado al cliente, eliminando por completo los cuadernos y las anotaciones perdidas.

---

## 🎯 ¿Qué problema resuelve BITs?

> **El control total de tu taller en un solo lugar:** Con BITs sabés exactamente qué le pasa a cada equipo, qué técnico lo está revisando, qué repuestos se usaron, cuánto tiempo llevó el trabajo y cuál es el estado de la reparación en tiempo real para mantener siempre informado al cliente.

---

## 🚀 ¿Qué podés hacer con el sistema? (Funcionalidades)

### 👥 Gestión Integral de Clientes
Crea una base de datos unificada con las personas que confían en tu servicio:
* **Altas rápidas:** Registrá nuevos clientes con sus datos esenciales (Teléfono, Dirección, Correo electrónico).
* **Notas personalizadas:** Agregá observaciones importantes sobre el cliente *(ej: "Llamar solo por la tarde")*.
* **Control de estados:** Visualizá rápidamente qué clientes están activos y quiénes de baja.

### 💻 Control Detallado de Equipos y Hardware
Evitá confusiones y mantené la transparencia absoluta sobre lo que recibís:
* **Historial por cliente:** Vinculá múltiples computadoras a un mismo dueño.
* **Ficha Técnica Específica:** Al ingresar una **PC o Notebook**, el sistema te permite registrar su procesador, memoria RAM, almacenamiento (disco) y estado de la pantalla para saber exactamente qué especificaciones tiene el equipo antes de abrirlo.

### 📝 Bitácora de Órdenes de Trabajo (OT)
El corazón del taller. Controlá el flujo de trabajo de manera cronológica:
* **Ingreso ordenado:** Generá una orden detallando la falla exacta que reporta el usuario.
* **Fechas automáticas:** El sistema congela la fecha y hora exacta de recepción para medir tus tiempos de respuesta.

### ⏱️ Línea de Tiempo de Tareas Realizadas
Documentá paso a paso el progreso técnico:
* **Historial clínico:** Registrá cada acción realizada *(ej: "Limpieza de coolers", "Instalación de Sistema Operativo")*.
* **Tiempos de trabajo:** Medí cuánto tiempo te llevó cada tarea específica.
* **Estados del servicio:** Cambiá el estado del equipo a *En Diagnóstico, Esperando Repuestos o Listo para entregar* para que todo tu equipo esté coordinado.

### 🔌 Control de Repuestos y Costos
Evitá pérdidas económicas y asegurá la rentabilidad:
* **Inventario aplicado:** Anotá qué componentes invertiste en el arreglo *(ej: "Disco SSD 480GB", "Pasta térmica")*.
* **Cálculo transparente:** Asigná el costo de cada repuesto para saber con precisión los gastos de la reparación.

## 🛠️ Tecnologías Detrás del Sistema

Para lograr una aplicación **rápida, liviana y muy segura** que no requiera computadoras potentes ni modernas para funcionar en el taller, BITs utiliza:

* **Entorno Visual:** Delphi 7 (Garantiza un programa rápido que arranca al instante).
* **Motor de Datos:** PostgreSQL (Tus datos protegidos en una base de datos profesional).
* **Conector:** ZeosLib (Asegura que el programa y la base de datos hablen de forma fluida y sin microcortes).

---

## 💡 Ventajas Clave para tu Negocio

1. **Transparencia absoluta:** Ante cualquier duda de un cliente, tenés la ficha de hardware exacta con la que ingresó el equipo.
2. **Historial clínico de la PC:** Si un equipo vuelve meses después, podés revisar la bitácora anterior para ver qué falla tuvo y qué componentes se le cambiaron.
3. **Mayor orden:** Identificá al instante qué equipos urgen repararse y cuáles están listos para retirar, mejorando la satisfacción del cliente.
